# ArduinoPCMonitoring
Show PC hardware information on Arduino with OLED display

## Feature

* CPU Temp
* CPU Fan RPM
* GPU Temp
* GPU Fan RPM


## Use Components

* Arduino Uno R3 (can replace other Embedded SBC)
* 0.91' OLED Display (I2C)
* USB cable (for Serial Communication & Power)

## Library

* OpenHardwareMonitoring (OHM)
